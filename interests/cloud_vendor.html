Cloud Computing

Info on Cloud Computing:
Cloud computing is an information technology (IT) paradigm that enables ubiquitous access to shared pools of configurable system resources and higher-level services that can be rapidly provisioned with minimal management effort, often over the Internet. Cloud computing relies on sharing of resources to achieve coherence and economy of scale, similar to a utility.


Uses of Cloud Computing:

  • Data Storage
  • Mobility:
    • Applications like Google Drive allow access to data that is stored on the cloud rather than just stored on a PC.
  • Backup:
    • Cloud allows for ease of backing up data which in the past was a time consuming and frustrating task.


Definition of Cloud Computing: the practice of using a network of remote servers hosted on the Internet to store, manage, and process data, rather than a local server or a personal computer.
